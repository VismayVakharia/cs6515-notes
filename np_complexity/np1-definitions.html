Complexity Classes
...

NP = class of all search problems where we can efficiently verify solutions in polynomial time
(almost same as the class of all decision problems)
P = class of all search problems that are solvable in polynomial time (generate)

    • it is as difficult to solve a problem as to verify a solution
    • Proofs: generating a proof for a problem is as difficult as to check a given proof
    • If verifying can be done in polynomial time then solving can also be done in polynomial time
    • it is easier to verify a solution compared to generate one

...

Input: boolean formula in CNF form with variables and clauses
Output: satisfying assignment of variables if one exists, otherwise

Run-time to verify an assignment satisfies the CNF is

SAT NP

NP-completeness
...

If P != NP, there are some search problems that can not be solved in polynomial time -> NP-complete problems

NP-complete problems are the hardest problems in the class NP
if P != NP then all NP-complete problems are not in P

Contra-positive:
if there exists a NP-complete problem can be solved in polynomial time -> all problems in NP can be solved in polynomial time => P = NP

SAT is NP-complete
...

  1. SAT NP
  2. if we can solve SAT in polynomial time then we can solve every problem in NP in polynomial time

if P != NP, then SAT P

Reduction
...

Problems A and B (example: A=colouring & B=SAT)
Reduction: A -> B or A <= B
reducing A to B, if we can solve problem B in polynomial time the we can use that algorithm to solve A in polynomial time.

Colouring SAT
...

Suppose there is a polynomial time algorithm for SAT and use it to get a polynomial time algorithm for Colouring.

  • f: input for colouring (G, k) -> input for SAT f(G, k)
  • h: solution S for f(G, k) -> solution for colouring h(S)
  • Need to prove
    • S is a solution to f(G, k) <=> h(S) is a solution to (G, k)

For example: To show that independent set (IS) is NP-complete

  1. IS NP
  2. for all A NP, A IS

Simplification
...

  • Assume we know that SAT is NP-complete (known)
  • Thus, for all A NP, A SAT
  • Suppose we show SAT IS
  • Then, A SAT IS A IS
  1. IS NP
  2. known NP-complete problem: SAT IS
